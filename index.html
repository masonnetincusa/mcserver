What is this server about?

This server is about everything honestly. This is a chill server. You can chill out or do whatever you want, even having the possibility of earning creative mode access! You start out in survival, and you decide where you want to go from there! You can call this a fun server, I guess.

It can happen repeatedly and can get very annoying after a while, but that is because we need to upgrade the server, which we will when we have a higher player count. The max player count is 1,000,000, but the server can't handle that many players at the same time right now.
